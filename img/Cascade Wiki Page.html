Cascade

The cascade is a fundamental feature of CSS. It is an algorithm defining how to combine properties values originating from different sources. It lies at the core of CSS as stressed by its name: Cascading Style Sheets.

Example

If you apply a class to an html tag, the affects of that tag will cascade to all elements nested within that tag.
 <div class="style">
    <p>You can put any kind of content here. Such as <p>, <table>, etc.<p>
</div>
.style {
  background-color: green;
  color: white;
}
This will display on the page as:

You can put any kind of content here. Such as <p>, <table>, etc.
